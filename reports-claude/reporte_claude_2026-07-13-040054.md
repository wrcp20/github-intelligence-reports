## Highlights del Día

- **Claude Code v2.1.207 lanzado el 11 de julio**: Auto mode ya disponible sin opt-in en Bedrock, Vertex AI y Foundry. Se corrigieron múltiples bugs críticos en terminal, sesiones remotas, git worktrees y el agent view. Es la versión más reciente y estable.

- **MCP 2026-07-28 Release Candidate**: La especificación MCP más grande desde su lanzamiento introduce arquitectura stateless, MCP Apps (UIs interactivas), extensión Tasks para trabajos de larga duración, y autorización Enterprise-Managed (EMA) estable con SSO zero-touch vía Okta, Anthropic, VS Code.
- **SDK TypeScript v0.111.0 añade soporte para "dreaming"**: Lanzado el 10 de julio, este release añade la capacidad `dreaming` a la API y mejora el control de sesiones de herramientas con `evaluated_permission`.

2. **Auto Mode como apuesta central**: La decisión de hacer Auto Mode disponible por defecto en Bedrock, Vertex AI y Foundry (v2.1.207) sin necesidad de opt-in señala la madurez percibida de esta función. Simultáneamente, el cambio del modo "default" a "Manual" en la CLI refleja una estrategia de seguridad más conservadora para usuarios CLI vs. API.
