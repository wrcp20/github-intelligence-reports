# Reporte Ecosistema Claude - 2026-06-09-200618

## Highlights del Día

- **🚀 Lanzamiento de Claude Fable 5 y Claude Mythos 5**: Hoy Anthropic lanzó `claude-fable-5`, su modelo más capaz disponible para el público general, junto a `claude-mythos-5` (exclusivo para participantes del Proyecto Glasswing). Ambos soportan ventana de contexto de 1M tokens por defecto, 128k max output y adaptive thinking siempre activo.
- **🔒 Nueva mecánica de rehúso y fallbacks**: La API ahora devuelve `stop_reason: "refusal"` cuando los clasificadores de seguridad rechazan una solicitud antes de generar output (sin cobro). El parámetro `fallbacks` (en beta) permite redirigir automáticamente las solicitudes rechazadas a otro modelo.
- **📦 Oleada masiva de releases SDK**: Todos los SDKs oficiales (Python v0.109.0, TypeScript v0.104.0, Agent SDK Python v0.2.95, Agent SDK TypeScript v0.3.170) actualizados hoy con soporte para los nuevos modelos Fable 5 y Managed Agents.
- **🛠️ Claude Code v2.1.170 publicado**: Incluye soporte para Claude Fable 5, corrección del bug de guardado de transcripts en terminal VS Code, y las nuevas flags `--safe-mode` y `/cd` del día anterior.
- **📋 MCP Inspector v0.22.0**: El inspector oficial del Model Context Protocol migró a publicación OIDC trusted en npm, mejorando la cadena de suministro de la herramienta.
